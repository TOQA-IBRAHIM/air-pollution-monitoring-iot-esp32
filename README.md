# Air Pollution Monitoring System (IoT – ESP32)

## 📌 Overview
This project presents an IoT-based air pollution monitoring system that collects environmental data using sensors and transmits it to the cloud for real-time visualization and analysis.

The system simulates a real-world smart monitoring solution for tracking air quality and environmental conditions.

---

## 🎯 Objective
To design and implement a smart IoT system that monitors air pollution and temperature, and visualizes the data remotely in real time.

---

## ⚙️ System Architecture
The system is built using an ESP32 microcontroller connected to sensors and cloud services.

Workflow:
1. Sensors collect environmental data  
2. ESP32 processes the data  
3. Data is transmitted via Wi-Fi  
4. Cloud platform stores and visualizes the data  

---

## 🧪 Sensors Used
### MQ-2 Gas Sensor
- Detects smoke and combustible gases  
- Outputs analog signal representing gas concentration  

### NTC Temperature Sensor
- Measures ambient temperature  
- Used to analyze the effect of temperature on gas readings  

---

## 🌐 IoT Platform & Simulation
- Initial design tested on TinkerCAD  
- Final implementation done using Wokwi (supports ESP32 and sensors)  
- ESP32 used for wireless communication and data transmission  

---

## ☁️ Cloud Integration
- Data transmitted using ESP32 Wi-Fi module  
- Connected to cloud platform (ThingSpeak-compatible)  
- Enables remote monitoring and real-time data access  

---

## 📊 Data Visualization
- Line Plot (Gas concentration over time)  
- Line Plot (Temperature over time)  
- Scatter Plot (Gas vs Temperature correlation)  
- Dual-axis plot for combined analysis
